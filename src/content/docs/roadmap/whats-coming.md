ruumy is being built in focused phases. Here's where things stand.

## Phase 1 — MVP (In progress)

The core of ruumy. Everything needed to make a room feel alive.

- ✅ Room creation with unique URL
- ✅ Join via link (no account required)
- ✅ Real-time chat
- ✅ Live user presence (who's in the room)
- ⬜ Deploy to production

## Phase 2 — Canvas & Identity (Next)

Making rooms feel more like spaces.

- ⬜ Live cursors on canvas
- ⬜ Shared notes editor
- ⬜ Post-it notes
- ⬜ Persistent display names (localStorage)
- ⬜ Avatar colors & initials

## Phase 3 — Embeds & History (Planned)

Making rooms more useful.

- ⬜ YouTube embed (Watch Party mode)
- ⬜ Image & link sharing on canvas
- ⬜ Room history (last 24h)
- ⬜ Whiteboard with basic drawing tools
- ⬜ Screen share (WebRTC)

## Phase 4 — Platform (Future)

Turning ruumy into a platform.

- ⬜ Room templates
- ⬜ Public rooms directory
- ⬜ API access
- ⬜ Plugin / widget system
- ⬜ AI assistant inside rooms

---

:::note
This roadmap reflects current thinking and will evolve. If you have a feature request, open an issue on [GitHub](https://github.com/ruumy-app).
:::
